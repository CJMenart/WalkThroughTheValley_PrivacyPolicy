# WalkThroughTheValley PrivacyPolicy
Automatically Collected Information 

While you are using the Application, it will use GPS technology and other location services to determine your current location. 

This information is only used to run the game. It never leaves your device. It is not visible to us, the developers, and it is not, nor will it ever be, shared with any third parties. Some location information is saved by the Application on your device, in the form of save files. You can remove this information from your device at any time by deleting those save files through the Application.


Your Consent

By using the Application, you are consenting to the processing of your information as set forth in this Privacy Policy.
